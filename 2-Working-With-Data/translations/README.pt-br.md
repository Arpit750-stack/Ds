Nestas aulas você irá aprender algumas das formas que dados podem ser geridos, manipulados e usados em aplicações. Você irá aprender sobre bancos de dados relacionais e não-relacionais, e como dados podem ser armazenados neles. Você irá aprender, também, os fundamentos de se trabalhar com Python para manipular dados, e você irá descobrir algumas das diversas formas que você pode trabalhar com Python para administrar e minerar esses dados.
